SOBRE O PROJETO

O projeto é uma página dinâmica criada utilizando apenas HTML, CSS e JavaScript vanilla, ou seja, sem o uso de frameworks ou bibliotecas externas. Ele é um exemplo de Single Page Application (SPA) e possui diversas funcionalidades, como a possibilidade de adicionar novos livros, categorias aos livros, visualizar os livros.

A página possui uma interface de usuário amigável e intuitiva, com um menu de navegação que permite ao usuário acessar as diferentes seções da aplicação. A seção de adição de livros inclui um formulário onde o usuário pode preencher os campos necessários, incluindo o título do livro, o autor, a categoria e a capa do livro em formato de imagem. Ao enviar o formulário, os dados são validados e adicionados à lista de livros.

O projeto é uma ótima oportunidade para aprender como criar uma SPA usando apenas as tecnologias básicas da web. Ele pode ser utilizado como base para projetos mais complexos e sofisticados, e também como um excelente exercício de programação e desenvolvimento web.
